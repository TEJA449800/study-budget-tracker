# 🎓 Smart Study-Budget Tracker
**Created by M. Surya Teja**

## 🌟 Overview
A Python-based web application built with Streamlit to help Class 10 students balance their academic goals with their daily expenses.

## 🚀 Features
- **Interactive Sidebar:** User-friendly inputs for study hours and spending.
- **Dynamic Metrics:** Real-time calculation of study efficiency compared to Board Exam targets.
- **Visual Feedback:** Progress bars, custom alerts, and celebration balloons on success.

## 💻 Tech Stack
- **Language:** Python 3.x
- **Framework:** Streamlit
- **Logic:** Custom conditional algorithms for academic & financial tracking.
